
# Hawaiian Word Pronouncer
### Hawaiian words can be intimidating to attempt to pronounce.
###### The 12 valid characters in the Hawaiian language are:
a, e, i, o, u, p, k, h, l, m, n, and w. 
###### 
p, k, h, l, m, n pronounced like the english versions.
###### 
w -start of word pronounced as a w sound. w -after ‘i’ or ‘e’ pronounced as a v sound, otherwise it is pronounced as w sound.
#####
- a sounds like ah
- e sounds like eh
- i sounds like ee 
- o sounds like oh 
- u sounds like oo 
- ai sounds like eye
- ae sounds like eye
- ao sounds like ow
- au Sounds like ow
- ei Sounds like ay
- eu Sounds like eh-oo
- iu Sounds like ew
- oi Sounds like oyo
- ou Sounds like ow
- ui Sounds like ooey
